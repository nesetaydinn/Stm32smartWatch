# Stm32smartWatch

Developer Neşet Aydın

The project developing for Graduation project,
Kernel: FreeRtos based cmsis-os
Gui lib: LittlevGl v6
Mcu: Stm32f407vg
Screen driver: st7789 & 240x240 px ips screen
